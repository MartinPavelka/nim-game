# nim-game
A competitive game of nim with either 3 or 4 piles for two players.

### Running Nim game

Go to your nim folder in Terminal:
```sh
$ cd /YOUR-NIM-FOLDER/java
```

Compile and run as java -jar:
```sh
$ java -jar nim.java
```
